dottedmag.net :: HTTPIZE 
Whenever a service requires you to provide a HTTPS endpoint
and you don't fancy runinng another tool during development.

GET https://dottedmag.net/httpize/8080/oauth/redirect?code=42
=>
HTTP 301 http://127.0.0.1:8080/oauth/redirect?code=42


To host HTTPIZE yourself put the following into Nginx config:

location ~* /httpize/.+ {
  limit_except GET HEAD {
    deny all;
  }
  rewrite ^/httpize/([0-9]+/.*)$ http://127.0.0.1:$1 permanent;
}
